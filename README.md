# 🌐 Epic Craft - Official Portal

This is the source code for the official **Epic Craft Mod** landing page. A modern, where players can find the latest downloads, project information, and meet the development team.


---

## 🚀 Features

* **Responsive Design:** Fully optimized for mobile and desktop using Bootstrap 5.
* **Fast Navigation:** Clean and intuitive interface for quick access to mod files.
* **Technical Integration:** Clear compatibility requirements (Forge 1.20.1) and version alerts.
* **Direct Links:** Integration with CurseForge for downloads and GitHub for documentation.

---

## 🛠️ Built With

* **HTML5 / CSS3**
* **Bootstrap 5** (Dark Theme)
* **Bootstrap Icons**
* **JavaScript** (For dynamic page loading/routing)
